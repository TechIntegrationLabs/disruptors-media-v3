## 🚀 **Quick Start Guide**

### For Local Development (100% Features)
```bash
git clone [repo-url]
cd disruptors-media-v3
npm install
npm run dev:full          # Complete environment with Claude Code + Railway MCP
# Triple-click logo → type "admin" to access intelligent admin panel
```

### For Railway Production Deployment (95% Features)
```bash
npm run deploy:railway    # Full-stack deployment with WebSocket support
# Use Claude Code: "Check Railway deployment status"
```

### For Public Marketing Site (Frontend Only)
```bash
npm run deploy:netlify    # Static hosting with read-only admin panel
```
